# Johannsen-Psaltis metric

The Johannsen–Psaltis metric is a Kerr-like parametric deviation designed to test the no-hair
theorem in the strong-field regime close to a black hole.

## Metric definition

The  Johannsen–Psaltis metric is given by:



$$ds^2 =\;&
- \left(1 - \frac{2Mr}{\Sigma}\right)\left(1 + h\right)\, dt^2
- \frac{4Mar \sin^2\theta}{\Sigma}\left(1 + h\right)\, dt\, d\phi \\
&+ \frac{\Sigma\left(1 + h\right)}{\Delta + a^2 h \sin^2\theta}\, dr^2
+ \Sigma\, d\theta^2 \\
&+ \sin^2\theta
\left[
r^2 + a^2
+ \frac{2Ma^2 r \sin^2\theta}{\Sigma}
+ h\, \frac{a^2(\Sigma + 2Mr)\sin^2\theta}{\Sigma}
\right] d\phi^2$$


where  

$$\Sigma = r^{2} + a^{2} \cos^{2}\theta, \quad
\Delta = r^{2} - r_{s} r + a^{2}, \quad
r_{s} = 2 M$$

- ($t$) : time coordinate
- ($r$) : radial coordinate  
- ($\theta$) : polar angle  
- ($\phi$) : azimuthal angle  
- ($a$) : spin parameter $(0 \le a \le M)$  

When $(a = 0)$, this reduces to the Schwarzschild metric.
