## Johannsen–Psaltis metric

The Johannsen–Psaltis metric is a Kerr-like parametric deviation designed to test the no-hair
theorem in the strong-field regime close to a black hole.

## Metric definition

The Johannsen–Psaltis metric is given by

$$ds^2 = 
- \left(1 - \frac{r_s r}{\Sigma}\right)(1+h)\, dt^2

- \frac{2 r_s a r \sin^2\theta}{\Sigma} (1+h)\, dt\, d\phi + \frac{\Sigma (1+h)}{\Delta + a^2 h \sin^2\theta}\, dr^2

+ \Sigma\, d\theta^2 \\

+ \left(r^2 + a^2 + \frac{r_s a^2 r \sin^2\theta}{\Sigma} \\


+ \frac{a^2 (\Sigma + r_s r) \sin^2\theta}{\Sigma} h \right) \sin^2\theta\, d\phi^2$$

$$\Sigma = r^2 + a^2 \cos^2\theta, \quad \Delta = r^2 - r_s r + a^2$$

$$h = \epsilon_3 \frac{(r_s/2)^3 r}{\Sigma^2}$$
