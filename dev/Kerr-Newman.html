Kerr - Newman metric

The Kerr-Newman metric describes the geometry of spacetime outside an electrically charged rotating black hole

Metric definition

The Kerr-Newman metric is given by:

\[ds^2 = -\left(1 - \frac{r_{s}r- Q^{2}}{\Sigma}\right)dt^2 - \frac{2a (r_{s}r - Q^2) \sin^{2}\theta}{\Sigma}dtd\phi + \frac{\Sigma}{\Delta}dr^{2} + \Sigma d\theta^{2} + \left(r^2 + a^2 + \frac{(r_{s}r - Q^2)a^{2}\sin^2\theta}{\Sigma}\right)\sin^2\theta d\phi^2\]

or in matrix form:

\[g_{\mu\nu} = \begin{pmatrix} -\left(1 - \dfrac{r_s r - Q^2}{\Sigma}\right) & 0 & 0 & -\dfrac{2a (r_{s}r - Q^2) \sin^{2}\theta}{\Sigma} \\ 0 & \dfrac{\Sigma}{\Delta} & 0 & 0 \\ 0 & 0 & \Sigma & 0 \\ -\dfrac{2a (r_{s}r - Q^2) \sin^{2}\theta}{\Sigma} & 0 & 0 & \left(r^2 + a^2 + \dfrac{(r_{s}r - Q^2)a^{2}\sin^2\theta}{\Sigma}\right) \sin^2\theta \end{pmatrix}\]

where

\[\Sigma = r^{2} + a^{2} \cos^{2}\theta, \quad \Delta = r^{2} - r_{s} r + a^{2} + Q^{2}, \quad r_{s} = 2 M\]

  • ($t$) : time coordinate
  • ($r$) : radial coordinate
  • ($\theta$) : polar angle
  • ($\phi$) : azimuthal angle
  • ($a$) : spin parameter $(0 \le a \le M)$
  • ($Q$) : Electric charge

When $ (Q = 0)$, this reduces to the Kerr metric.
